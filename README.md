# SafeCam - ASP.NET MVC Application

## About the Project
SafeCam is a dynamic web application built to demonstrate practical skills using the ASP.NET MVC framework. The project includes well-structured routing, controllers, and views to create a user-friendly experience.

##  Technologies Used
* **Framework:** ASP.NET MVC
* **Language:** C#
* **Styling:** SCSS, CSS, HTML

##  Key Features
* Implementation of the Model-View-Controller (MVC) architectural pattern.
* Server-side rendering for dynamic content.
* Clean and responsive UI design.
